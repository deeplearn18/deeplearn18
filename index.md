## 法律法规

```
系统分解、碎片拼图；
概念原理、理论实务；
融会贯通、价值创造；
天道酬勤、量变质变。
```
> ——闻道有先后，术业有专攻，兴趣是最好的老师。

> 互为一体，向下兼容...

```
1. 储备（布局、策略性申请等）与风控（专利、商标及版权等）
2. 储备与运营——价值创造
3. 前端与中后端
4. SEP与非SEP
5. 发明人、IPR及代理人
6. 体系系统、团队建设、项目管理、执行闭环
7. 格局、节奏及温度（高度，温度，风度）
8. 自己，家人，他人；小我、大我到无我。
```
